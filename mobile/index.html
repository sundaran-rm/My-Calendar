Connect to Firebase
Firebase lets you sync across devices and collaborate in real-time using a shared room code — no accounts needed.
1
Create a Firebase project
Go to console.firebase.google.com → Add project → disable Analytics → Create.
2
Add a Web app & copy config
Project Overview → </> Web → register → copy the firebaseConfig object.
3
Enable Firestore
Build → Firestore Database → Create → test mode → any region → Enable.
4
Paste your config
Paste exactly as shown — the full const firebaseConfig = { … } block works fine.
